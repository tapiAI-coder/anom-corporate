## 設計判断

- **ビルド工程なしの静的サイト**（npm不要）— テキスト修正→公開まで自走できる最小複雑度。ブラックボックス化させない
- **ライブラリは自前ホスト**（`assets/vendor/`、バージョン固定）— CDN障害・改ざんリスクを排除
  - Three.js r128 ／ GSAP 3.12.5 ／ ScrollTrigger 3.12.5 ／ Lenis 1.1.14（いずれも試作で動作検証済みのバージョン）
- **全て相対パス** — github.io配下でも独自ドメイン直下でも同じコードで動く

## フォルダ構成（新）

```
anom-corporate/
├── index.html          ← 本編（1ページ）
├── privacy.html        ← プライバシーポリシー
├── 404.html            ← 迷子ページ
├── robots.txt / sitemap.xml
├── assets/
│   ├── css/style.css   ← 全スタイル
│   ├── js/             ← 役割別に分割（config / particles / animations / form / main）
│   ├── vendor/         ← 同梱ライブラリ（バージョン固定・触らない）
│   └── img/            ← ファビコン・OGP・ロゴ
├── docs/               ← この計画書・運用マニュアル
└── CLAUDE.md           ← 技術司令塔（静的サイト版に刷新）
```
